# Markdown

Ctrl+Alt+O: Preview Markup in Browser

Ctrl+Alt+X: Export Markup as HTML

# Goto Anything功能 — 快速查找ctrl + P

输入@+函数名可以快速找到函数

输入#+文本可以快速进行文件内文本匹配

# 多行游标功能（ctrl + D，非常实用）

  1. 利用查找替换功能：ctrl + H
  2. 多行游标功能（推荐）：ctrl + D 如何将文件中的某个单词更改为另一个？ 选中一个单词后，按ctrl+D可以同时选中文档中其他相同的该单词，同时每个单词后面都多一些光标如果有某些不想被修改的单词则按ctrl+K，这样ctrl+D就会跳过这个进入下一个符合条件的模式行然后ctrl+shift+L即可修改

# 使用小技巧

  1. 快速跳转到某行：Ctrl+G，再输入行号
  2. 快速添加新行： Ctrl+Enter：在当前行下方新建一行 Ctrl+Shift+Enter：在当前行上方新建一行
  3. 多行缩进：Tab和Shift+Tab或者选中多行后按Ctrl+]可以增加缩进，按Ctrl+[可以减少缩进
  4. 带格式的拷贝：从其他地方拷贝至SublimeText的时候使用Ctrl+Shift+V
  5. 重新打开关闭的标签：Ctrl+Shift+T 多次按的话会按你关闭的顺序来重新打开
  6. 上下移动行：选中后，Ctrl+Shift+↑↓可以上下移动
  7. Ctrl+Shift+D：可快速复制光标所在的行到该行之前
  8. Ctrl+Shift+M：选中花括号里面的全部内容不包括{}
  9. Ctrl+Shift+K：删除整行
  10. 代码补全：比如写标签的时候，按下Alt+. 即可得到

# 选择类

  1. Ctrl+D 选中光标所占的文本，继续操作则会选中下一个相同的文本
  2. Alt+F3 选中文本按下快捷键，即可一次性选择全部的相同文本进行同时编辑
  3. Ctrl+L 选中整行，继续操作则继续选择下一行，效果和 Shift+↓ 效果一样
  4. Ctrl+Shift+L 先选中多行，再按下快捷键，会在每行行尾插入光标，即可同时编辑这些行
  5. Ctrl+M 光标移动至括号内结束或开始的位置
  6. Ctrl+Shift+A 选择光标位置父标签对儿
  7. Ctrl+Shift+M 选择括号内的内容（按住-继续选择父括号）

# 编辑类

  1. Ctrl+J 合并选中的多行代码为一行
  2. 删除整行：Ctrl+K+K 从光标处开始删除代码至行尾Ctrl+Shift+K 删除整行
  3. 注释：Ctrl+/ 注释单行 Ctrl+Shift+/ 注释多行
  4. 大小写：Ctrl+K+U 转换大写Ctrl+K+L 转换小写
  5. 撤销与恢复撤销：Ctrl+Z 撤销Ctrl+Y 恢复撤销
  6. Ctrl+K Backspace 从光标处删除至行首

# 搜索类

  1. 搜索： Ctrl+F 打开底部搜索框，查找关键字 Ctrl+shift+F 在文件夹内查找，与普通编辑器不同的地方是sublime允许添加多个文件夹进行查找，略高端，未研究
  2. 项目搜索： Ctrl+P 打开搜索框举个栗子： 输入当前项目中的文件名，快速搜索文件， 输入@和关键字，查找文件中函数名， 输入：和数字，跳转到文件中该行代码， 输入#和关键字，查找变量名
  3. 跳转到某行： Ctrl+G 打开搜索框自动带：输入数字跳转到该行代码 Ctrl+R 打开搜索框，自动带@，输入关键字，查找文件中的函数名 Ctrl+：打开搜索框，自动带#，输入关键字，查找文件中的变量名、属性名等 Esc 退出光标多行选择，退出搜索框，命令框等

# 显示类

Ctrl+Tab 按文件浏览过的顺序，切换当前窗口的标签页 Ctrl+PageDown 向左切换当前窗口的标签页 Ctrl+PageUp 向右切换当前窗口的标签页 Alt+Shift+1 窗口分屏，恢复默认1屏（非小键盘的数字） Alt+Shift+2 左右分屏-2列 Alt+Shift+3 左右分屏-3列 Alt+Shift+4 左右分屏-4列 Alt+Shift+5 等分4屏 Alt+Shift+8 垂直分屏-2屏 Alt+Shift+9 垂直分屏-3屏 Ctrl+K+B 开启/关闭侧边栏 F11 全屏模式 Shift+F11 免打扰模式
